# Chabot AI Flutter

Integrated Chatbot App by Gemini AI built with Flutter

<img src="./screenshoot/ss.png">

## Installer
```
git clone https://github.com/fitri-hy/chatbot-ai-flutter.git
cd chatbot-ai-flutter
flutter pub get
flutter run
```

## Required
- Flutter 3.22.2
- SDK 34.0.0
- JDK 19
- Java 22.0.1
